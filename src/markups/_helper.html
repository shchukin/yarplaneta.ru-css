.helper

A simple line of helping text. It has the top padding set, and it meant to be used with components such as .input:

Can’t be changed later

Notice that helper is placed after input (not inside). It makes helper component universal. So it can be used with any other element:

Random element
Changes will be saved automatically



.helper--error

Can be used for outputting error message. A few in a row is also possible:

Password is too short
Password must contain a special character
Password must be hard to guess: dates, phone numbers and names are not allowed and either it shouldn’t be based on public data
New password is similar to previous password
